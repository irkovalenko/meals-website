# React + Vite + Tailwindcss

Project created as exercises in interaction with API and React.js. Used [TheMealDB](https://www.themealdb.com/) and by following [The Codeholic React Crash Course](https://www.youtube.com/watch?v=GPWHql7scWc&t=8777s).

All data is coming from API calls.

The project consists of 3 NavLinks : 
* **Home** - displays random 10 meals if user does not type anything in search or the relevant meals if user types something in search. Each meal card is clickable so user can access the meal detail page (recipe, basically).
